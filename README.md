# 智慧图书管理系统简介 java90

## 项目概述


## 技术栈

- **后端：** 基于Spring Boot框架，运用Java 1.8版本
- **前端：** Vue2技术，搭配HTML页面展示
- **数据库：** 数据持久层使用MyBatis-Plus操作MySQL数据库

## 功能模块

### 系统角色

- **管理员**
  - 图书信息全管理：添加、编辑、删除
  - 库存管理
  - 用户信息管理：注册审核、权限分配、查看借阅记录
- **用户**
  - 图书搜索与查看
  - 借阅、续借、归还操作
  - 个人借阅历史与逾期情况查询

## 运行环境

- 后端：Java 1.8及以上版本环境
- 前端：现代浏览器环境
- 数据库：MySQL

## 部署步骤

1. 配置数据库环境
2. 部署后端服务
3. 部署前端应用

## 目录结构

```
项目目录
├── backend              后端代码
├── frontend             前端代码
└── database             数据库设计及脚本
```

## 核心亮点

- **分离式架构：** 前后端分离，便于维护与扩展
- **高效稳定：** Spring Boot与Java 1.8结合，保障系统性能
- **易用性：** 界面友好，功能完善，便于用户操作
- **完善文档：** 系统设计、数据库设计、功能说明详尽的文档支持
